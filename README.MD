# Spam Filter Using Naive Bayes

This is an implementation of the paper: Spam Filtering with Naive Bayes - Which Naive Bayes?

You can find the research paper on[Research Gate](https://www.researchgate.net/publication/221650814_Spam_Filtering_with_Naive_Bayes_-_Which_Naive_Bayes) with open access.

My implementation will be in Golang. The Data used is found in this [website](https://www2.aueb.gr/users/ion/data/enron-spam/).
